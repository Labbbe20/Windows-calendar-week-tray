# windows-calendar-week-tray

English

Week Number Tray is a lightweight Python application for Windows that displays the current calendar week directly in the system tray.

The application runs quietly in the background and allows users to see the current week number at a glance without opening a calendar or any additional software.

It was designed to provide a simple and convenient way to keep track of calendar weeks, especially in professional environments where week numbers are frequently used.

Deutsch

Week Number Tray ist eine kleine Python-Anwendung für Windows, die die aktuelle Kalenderwoche direkt im Infobereich der Taskleiste (System Tray) anzeigt.

Die Anwendung läuft unauffällig im Hintergrund und ermöglicht es, die aktuelle Kalenderwoche jederzeit auf einen Blick abzulesen, ohne einen Kalender oder eine zusätzliche Anwendung öffnen zu müssen.

Das Tool wurde entwickelt, um insbesondere im beruflichen Alltag eine schnelle und komfortable Übersicht über die aktuelle Kalenderwoche bereitzustellen.
